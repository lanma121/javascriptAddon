dom对象

document对象

区别：

js是document对象本身，jQuery包含0,context,lenght,_proto_属性

document[0],document.context与document的对象本身相等，都包含一些document本身具有的属性
lenght，代表document对象的数量
_proto_属性是document具有jQuery的属性和方法

jquery 中的$("dom")数组

1.包含每个元素自身具有的属性和方法，依次为0,1,2...数组
2.包含lenght属性
3.context属性是document对象
4.prevObject属性是document的jQuery对象
5.selector属性，jQuery选择器
6.__proto__属性，继承jQuery一些方法和属性

js 中的document.getElementsByClassName数组

1.包含每个元素自身具有的属性和方法，依次为0,1,2...数组,指定id者，直接有id的每个对象
2.包含lenght属性
3._proto_属性，继承HTMLCollection

js 中的document.getElementsByName数组

1.包含dom自身具有的属性和方法的每个元素，依次为0,1,2...排序
2.包含lenght属性，获取dom的数量
3.包含_proto_属性，继承NodeList
jquery 中的$("dom")，$(this)
1.本对象的jQuery对象
jquery 中的dom,this
1.本对象
